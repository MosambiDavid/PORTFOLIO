# Habitat Portfolio

Welcome to my portfolio project! This repository showcases my work in web development, databases, and server-side programming.

## 🚀 Features
- Responsive design with HTML, CSS, and JavaScript
- Database integration (SQLite)
- Node.js backend with Express
- Portfolio pages including resume and profile
- Custom modules for messaging and data handling

## 📂 Project Structure
- `index.html` – Main landing page
- `style.css` – Styling
- `script.js` – Frontend logic
- `server.js` – Backend server
- `db.js` – Database connection
- `resume.html` – Resume page
- `profile.png` – Profile image
